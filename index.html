rez'el (responsive element) documentation

rez'el offers two custom-elements, <r-grid> and <r-element> which switch between provided values in response to the user's @media.

<r-grid>

This all-purpose grid element automatically handles rows and cells, but can be fined tuned for each break-point

These cells are auto sized while these cells have specific widths at each break-point

<r-element>

This utility element enables you to apply specific style configurations at each breakpoint

text copy

resonsive attributes

for any supported attribute, one may provide a value for each break-point mobile tablet desktop xl-desktop. Omitted breakpoints will inherit from the last defined break-point. examples:
padding="medium" // medium padding for all break-points
margin-bottom="medium large" // medium bottom-margin on mobile, large for all others

<r-grid> attributes

  • text-align Any valid CSS text-align value
  • display Any valid CSS Display value (excluding multi-keyword syntax)
  • padding Dimensional values*
  • offset Number of columns to shift right
  • order Display order
  • font Font values*
  • col Number of columns to span
  • gap Space between elements, in Dimensional values

<r-element> attributes

  • text-align Any valid CSS text-align value
  • display Any valid CSS Display value, (excluding multi-keyword syntax)
  • padding Dimensional values*
  • offset Number of columns to shift right
  • margin Dimensional values*
  • font Font values*
  • margin-[side] top, left, bottom, right. Dimensional values*
  • padding-[side] top, left, bottom, right. Dimensional values*

Dimensional values

Note: Each non-zero value has [value]- counter-part with a negative value.
  • none 0px
  • small 4px
  • medium 8px
  • large 16px
  • xlarge 32px
  • xxlarge 64px

Font values

  • xsmall 12px
  • small 14px
  • medium 16px
  • large 24px
  • xlarge 32px
  • xxlarge 40px

Note: Dimensional and Font values can be overwritten

in your css, add definitions for --grid-dim-[value], or --grid-font-[value] 

:root {
  --grid-dim-xxlarge: 48px;
  --grid-dim-xxlarge-: -48px;  // negative counter-part
  --grid-font-xsmall: 14px;
  --grid-font-small: 18px;
  --grid-font-medium: 20px;
}

examples:
display="block none" // display mobile only
display="block none block" // hide from tablet

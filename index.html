rez'el (responsive element) documentation

rez'el offers two custom-elements, <r-grid> and <r-elm> which switch between provided values in response to the user's @media.

<r-grid>

This all-purpose grid element automatically handles rows and cells, but can be fined tuned for each break-point

These cells are auto sized while these cells have specific widths at each break-point

<r-elm>

This utility element enables you to apply specific style configurations at each breakpoint

text copy

resonsive attributes

for any supported attribute, one may provide a value for each break-point mobile tablet desktop xl-desktop. Omitted breakpoints will inherit from the last defined break-point. examples:
  • padding="medium" medium padding for all break-points
  • margin-bottom="medium large" medium bottom-margin on mobile, large for all others

<r-grid> attributes

  • skip-before skip the given columns before the element
  • skip-after skip the given columns after the element
  • text-align Any valid CSS text-align value
  • display Any valid CSS Display value (excluding multi-keyword syntax)
  • padding Dimensional-value*. Cannot accept shorthand
  • order Display order
  • font Font size in font-value*
  • col Number of columns to span
  • gap Space between elements, in dimensional-values*

<r-elm> attributes

  • text-align Any valid CSS text-align value
  • display Any valid CSS Display value, (excluding multi-keyword syntax)
  • padding Dimensional-value*. Cannot accept shorthand
  • margin Dimensional-value*. Cannot accept shorthand
  • font Font size in font-value*
  • margin-[side] top, left, bottom, right. dimensional-value*
  • padding-[side] top, left, bottom, right. dimensional-value*

Dimensional-values

  • none 0px
  • small 4px
  • medium 8px
  • large 16px
  • xlarge 32px
  • xxlarge 64px

Font-values

  • xsmall 12px
  • small 14px
  • medium 16px
  • large 24px
  • xlarge 32px
  • xxlarge 40px

Note: You can customize Dimensional and Font values

in your css, add definitions for --rezel-dim-[value], or --rezel-font-[value] 

:root {
  --rezel-dim-none: 0px;
  --rezel-dim-small: 4px;
  --rezel-dim-medium: 8px;
  --rezel-dim-large: 16px;
  --rezel-dim-xlarge: 32px;
  --rezel-dim-xxlarge: 64px;
  --rezel-font-xsmall: 12px;
  --rezel-font-small: 14px;
  --rezel-font-medium: 16px;
  --rezel-font-large: 24px;
  --rezel-font-xlarge: 32px;
  --rezel-font-xxlarge: 40px;
}
